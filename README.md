# FUSO

FUSO is written in Bash, and only in Bash. Initially, it didn't have a specific goal; it was simply a program where I honed my front-end and Bash skills. Even so, effort and time were spent to bring an ASCII draft to life. Incidentally, FUSO comes from PARAFUSO (screw).

Yes, I identify more as a front-end developer, so perfect and optimized code isn't what you should expect.

All translations by Google Translate.

## Installation

FUSO is a simple program, so downloading and extracting is all that's needed. But there is a minor dependency: a [Nerd Font](https://www.nerdfonts.com/font-downloads).

1. Clone the repository:
  ```sh
  git clone https://github.com/perrofaaias/fuso-sh.git
  cd fuso-sh
  ```
2. Complete the initial setup:
  ```sh
  bash bin/fuso.sh init
  ```
3. All set!
